# NexusCore Day 1 — SQL → SurrealDB (lesson)

This workspace is generated by `setup.sh`. It covers multi-model SurrealDB migration patterns, a shared connection pool, Wasm tenant boundaries, optional eBPF latency probes, and Prometheus metrics on port 9090.

Open the live dashboard at `http://127.0.0.1:9090/` after `./scripts/start.sh`, then run `./scripts/demo.sh` in another terminal to drive non-zero metrics.
