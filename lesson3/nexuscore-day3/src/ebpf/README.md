# eBPF (optional)

The original lesson narrative includes an XDP probe under this directory. On this track the
workspace focuses on the CSR blob format, native `nexuscore-graph` tests, and the demo host.

To experiment with BPF locally, add `xdp_tenant_router.bpf.c`, a `Makefile`, and build with
`clang` + `bpftool` as described in the course materials.
